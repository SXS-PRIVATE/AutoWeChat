# 微信工具
基于网页版微信完成的微信功能扩展
## 1、防撤回功能
- 语音
- 文字
- 图片
- 视频
- ...
## 2、自动回复功能
> 基于图灵机器人完成聊天自动回复
## 3、用户属性监测
> 动态监测用户属性编号，包括但不限于头像、昵称、签名...
## 4、群发消息、拉人进群、...
> 以上几个功能只是起到抛砖引玉的作用，请打开你的脑洞吧，朋友。
